## الخطوة 3: تثبيت اضافة Java

1. افتح VS Code
2. اضغط على أيقونة Extensions في الشريط الجانبي (`Ctrl+Shift+X`)
3. ابحث عن "Extension Pack for Java"

<img width="2559" height="1383" alt="image" src="https://github.com/user-attachments/assets/3749d3fb-3cb5-423f-9814-f611c1c243f5" />

4. اضغط "Install" على "Extension Pack for Java" من Microsoft
